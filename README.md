# Perlouic.com - Site Vitrine

Site vitrine pour **Perlouic**, réalisé avec **PicoCSS**.  
Ce site présente le quilt Oro, ses caractéristiques, la FAQ et les pages légales/contact.

---

## Structure du projet
index.html # Page d’accueil
quilt-oro.html # Boutique
faq.html # FAQ
contact.html # Contact
qu-est-ce-qu-un-quilt.html # Article de blog
mentions-legales.html # Mentions légales
conditions-generales-de-vente.html # CGV
assets/
css/
styles.css # Styles globaux
js/
product-gallery.js # Galerie produit et lightbox
img/
*.webp # Images produit et illustrations


---

## Instructions

1. Ouvrir `index.html` dans un navigateur moderne (Chrome, Firefox, Safari, Edge).  
2. Toutes les images sont optimisées en WebP.  
3. Les scripts JavaScript gèrent :
   - Galerie produit avec thumbnails et navigation
   - Lightbox sur la main image
   - Sélection de couleur et mise à jour des images
